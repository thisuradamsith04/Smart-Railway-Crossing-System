# 🚆 Smart Railway Crossing Automation System

## Overview

This project is a Smart Railway Crossing Automation System developed using Arduino Uno and Raspberry Pi 4.

The system automatically detects approaching trains, controls a railway gate, activates warning signals, displays real-time information on an LCD, and streams live camera footage through a Flask-based web dashboard.

---

## Features

✅ Automatic train detection using HC-SR04 Ultrasonic Sensor

✅ Automatic railway gate control using SG90 Servo Motor

✅ Traffic light signaling system

✅ Warning buzzer system

✅ LCD status display

✅ Live camera monitoring using Raspberry Pi Camera

✅ Flask-based web dashboard

✅ Real-time distance monitoring

✅ Train counting system

✅ Arduino ↔ Raspberry Pi serial communication

---

## Hardware Components

* Arduino Uno
* Raspberry Pi 4B
* HC-SR04 Ultrasonic Sensor
* SG90 Servo Motor
* I2C LCD Display
